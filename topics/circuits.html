How current and potential difference behave in different circuit types

What are Series and Parallel Circuits?

Series circuits have components connected end-to-end, so there is only one path for current to flow.
Parallel circuits have components connected across common points, creating multiple paths for current.

Left: Series circuit (one path)   |   Right: Parallel circuit (multiple paths)

Current in Series and Parallel Circuits

Series Circuit:

  • The current is the same at every point in the circuit.
  • If one component breaks, the whole circuit stops working.

Parallel Circuit:

  • The total current is shared between the branches.
  • Current in each branch depends on the resistance of that branch.
  • If one branch breaks, current can still flow in the others.

Potential Difference (Voltage) in Series and Parallel Circuits

Series Circuit:

  • The total potential difference (voltage) is shared between all components.
  • The sum of voltages across each component equals the total voltage supplied.

Parallel Circuit:

  • The potential difference across each branch is the same as the supply voltage.
  • Each branch receives the full voltage of the power source.
